# Osiris
Just a small cheat sheet for Monster Hunter games. Small icons for which monster you want to select and then a render on the monster card that when clicked will show weakness', recommended protections and drops. 

# Technologies Used
    -React/Frontend

    -Express/Backend

    -Vercel/Deployment
